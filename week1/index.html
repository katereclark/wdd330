Week 1 Notes and Exercises

Exercise(s)

  • Exercise 1: Story Writer

Questions

Notes

  • Design for smallest device first (mobile first)
  • Responsive retrofitting - addresses constraints of mobile size and bandwidth as opposed to simply making something work for mobile
  • Start small and scale up - content and functionality focus
  • Observe what people are doing rather than just what they are saying that they want
  • Immediate. Relevant. Frictionless.
  • Provide users with what they want as fast as possible to be useful
  • Make something 0 steps when possible
  • Take reachability into account - put key items at the bottom
  • Speed matters (even half a second)
  • Network is one of the greatest factors on speed
  • Reduce image usage (and use SVG and webfonts)
  • Optimize and minify CSS and JS files
  • Primary controls - move nav and subnav to bottom on mobile
  • Easy to reach primary controls leads to a perception of speed and improved engagement and satisfaction
  • Using the word 'menu' with a border around it is the most understandable form rather than a hamburger menu icon
  • Focus on the important things first and simple to decrease cognitive load when using forms
  • Only add info that is absolutely required and needed
  • Make fields taller in forms and put labels inside input box and use a float label that moves up in box when selected
  • Add progress indicator and enable autocomplete on forms
  • Do not mask passwords - show by default but add the option to hide it if they want
  • Autoadvance to next field and add spaces in form, especially with card information
  • Don't bring up standard keyboard if onyl need number input - use telephone keypad instead
  • Link phone numbers in anchor tag instead of just the text so people don't have to copy and paste it
  • Do not use pop-ups, especially on mobile sites
  • Make adds small and simple (thin until clicked)
  • Use ads that are underneath the content and are viewed through a mirror window as the user scrolls to read
  • Make ads appropriate for the given content and create empathy by connecting the two
  • Content first, ads second
